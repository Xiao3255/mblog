环境安装

  • yarn

# yarn

# 安装yarn(已安装nodejs)

curl -o- -L https://yarnpkg.com/install.sh | bash

根据提示看是否需要将设置Path

如

vim ~/.zshrc

写入以下代码，退出保存

export PATH="$HOME/.yarn/bin:$HOME/.config/yarn/global/node_modules/.bin:$PATH"

默认情况下会自动添加路径

让配置生效

source ~/.zshrc #if you config .zshrc

安装完查看是否安装成功

yarn --version
