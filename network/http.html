    # 网络相关

    # 浏览器发送请求过程

    # 简版

    • 查找本地HOST,查看是否有相应的缓存记录，如果没有请求DNS服务器，DNS解析域名返回ip地址
    • 通过ip地址找到相应的服务器，发起连接请求
    • 通过三次握手协议后，建立连接
    • 服务处理相应的请求后，返回数据
    • 通过四次握手协议断开连接
    • 浏览器做相应的处理

    # CORS，设置请求允许携带凭证

    当跨域时要设置cookie除了后端要正确的配置外

    Access-Control-Allow-Credentials:true
Access-Control-Allow-Origin: /* alloworigin.com */
setcookie( ... ,'alloworgin') //设置要设置cookie的域

    ajax也需要设置允许携带凭证

    xml = new XMLHttpRequest();
xml.withCredentials = true;
